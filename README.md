# Alfred AI Prompt Workflow

A workflow for Alfred that provides quick access to AI assistants like ChatGPT, Claude, Copilot, etc.

## Usage

Type `ask` followed by:
- `co`: GitHub Copilot
- `gpt`: ChatGPT
- `cl`: Claude
- `px`: Perplexity
- `ms`: Mistral

Example: `ask gpt how do I center a div?`
